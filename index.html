oclcsrw

A framework for putting an SRW/SRU interface in front of your local database

View on GitHub Download .zip Download .tar.gz

The OCLC SRW/SRU Open Source project offers software that implements both the SRW Web Service and the SRU REST model interface to databases. Parallel projects include support for DSpace, Lucene, OCLC's Pears and Newton databases and a Federated Searching framework.

General information about SRW/SRU can be found on the WikiHomePage

Documentation for installation and operation of the SRW/SRU server can be found on the InstallationInstructions page

Documentation about configuration files for the SRW/SRU server can be found on the ConfigurationFiles page

We support persistent URIs in an environment where URIs come and go all the time. A description of how we do that using Merged databases can be found at PersistentURIPattern
